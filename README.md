# Simple Warehouse Management

Программа для складского учета на с++

## Возможности

- Добавление нового товара
- Просмотр всех товаров в виде таблицы
- Сохранение товаров в файл `database.txt`
- Загрузка товаров из файла при запуске программы
- Простое меню в консоли

## Меню программы

```text
1 - Добавить товар
2 - Показать все товары
3 - Сохранить данные
0 - Выход
```

## Хранение данных

Данные сохраняются в файл `database.txt`.

Формат строки в файле:

```text
id|name|quantity|price
```

Пример:

```text
1|Milk|10|55.5
2|Bread|20|32
```

## Как собрать проект

Нужен компилятор C++ с поддержкой стандарта C++17.

Пример сборки через `g++`:

```bash
g++ -std=c++17 main.cpp -o swm
```
